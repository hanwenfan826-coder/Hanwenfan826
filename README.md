# 短视频编导智能体网站

这是一个可公开发布的静态网站，入口文件是 `index.html`。

## 发布到 GitHub Pages

1. 创建一个公开 GitHub 仓库，例如 `short-video-director-agent`。
2. 上传本文件夹中的所有文件到仓库根目录。
3. 在仓库 `Settings -> Pages` 中选择从默认分支发布，目录选择根目录。
4. 发布成功后，把 `robots.txt` 和 `sitemap.xml` 里的 `https://YOUR_PUBLIC_DOMAIN` 改成真实网站地址。
5. 到 Google Search Console 或百度搜索资源平台提交站点地图，加快收录。

## 搜索收录说明

公开发布后，网站可以被访问，但搜索引擎收录不是即时的。通常需要几天到几周，取决于域名权重、外链、提交情况和内容质量。
